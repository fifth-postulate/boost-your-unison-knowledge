# Challenges

The workshop is structered as a number of challenges. Some of the challenges
are [Unison canon][unison:canon] and can be found on their pages.

Other challenges teach you something about Unison that might be a new concept,
like:

* Unison Codebase Manager
* Content-addressable Code
* Effect handlers and abilities
* Documentation
* Testing

Some of those challenges lend themselves for demonstrations, which we tend to
give. Depending on the time we will give all demonstrations, or we could let
you decide what is most interesting.

For each challenge you should be able to find our solution, if you ever get
stuck. And feel free to ask questions.

[unison:canon]: https://www.unison-lang.org/docs/#learning-labs
