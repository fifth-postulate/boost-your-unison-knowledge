# Cloud Labs

[Unison Cloud][unison.cloud] is Unisons answer to microservices. It has a
ton of features, and a number of [labs][unison.cloud:labs] to get to know them.

Some of the labs are longer projects so might be better suited to take on after
the workshop.

We provide a transcript for `cloud-hello-world.u`.
