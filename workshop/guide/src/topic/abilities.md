# Abilities

Abilities are Unison's approach to [_Algebraic Effects_](https://en.wikipedia.org/wiki/Effect_system).

Algebraic effects is a way to describe what for an "effect", e.g. for example
side effects, a program can have.

This novel notion is described in the section `abilities` of our `booster-2026`
library. It is subdived into a number of parts, which build upon each other.
