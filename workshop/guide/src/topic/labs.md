# Learning Labs

The Unison documentation has a section called
[_learning labs_](https://www.unison-lang.org/docs/#learning-labs). Here
that inspired the last section; first glance.

Furthermore you will find the
[Wordle clone](https://www.unison-lang.org/docs/labs/wordle/docs/intro/)
challenge.

It is a great way to take a deep dive into Unison. Even though there are
various hints, tips and tricks, you still find a transcript in
`examples/wordle.u`.
