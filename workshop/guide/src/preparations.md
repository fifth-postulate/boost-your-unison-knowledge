# Preparations

In order to be able to fly through the material you could setup the your
development machine in the following way:

* Make sure to install Unison
* Setup your editor of choice
* Fetch the sources

## Installing Unison

In order to enjoy Unison, you first need to install it.

A complete [installation guide][how-to:install] can be found on the
documentation page.


## Setup your editor

Unison still uses files and you need an editor to edit them. There are some 
options how to [configure editors][how-to:editors] found on the Unison
documentation page.

## Download the sources

The sources for this guide, its examples and references can be found at [the
project repository][repo].

It also holds [a downloadable release][release]; a package of all aforementioned
items.
